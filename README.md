// ALESSANDRO ADAMI 2076065

Il progetto implementa un convertitore di numeri interi da arabi a romani, stampandoli poi in asciart

Il programma soddisfa i requisiti descritti a [questo indirizzo](https://it.wikipedia.org/wiki/Sistema_di_numerazione_romano)
